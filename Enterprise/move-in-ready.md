# Pronto per l'utilizzo

 **Sintesi:** in questo articolo viene descritto come visualizzare i dettagli relativi al set di opzioni di archiviazione cloud integrato nell'offerta cloud di Microsoft.
  
Soluzioni di archiviazione pronte per l'utilizzo:
  
- Utilizzare le soluzioni predefinite immediatamente
    
- Sono pronte all'uso e in dotazione con i servizi esistenti
    
- Forniscono personalizzazione limitata
    
Le sezioni seguenti descrivono i dettagli di ciascuna opzione di archiviazione "Pronto per l'utilizzo".
  
## SharePoint Online in Office 365

### Caratteristiche

- I file sono sicuri, affidabili e accessibili da più dispositivi e da qualsiasi luogo
    
- Il contenuto è facilmente individuabile tramite ricerca
    
- I flussi di lavoro e i criteri di conservazione semplificano l'utilizzo e la gestione del contenuto
    
- È possibile accedere al contenuto tramite le API web o un browser oppure non in linea
    
### Utilizzi comuni

- Portale aziendale o dell'organizzazione
    
- Collaborazione in team
    
- Ricerca contenuti organizzazione
    
### Scenari di archiviazione chiave

- Collabora
    
- Comunicazioni aziendali
    
- Gestire i video
    
### Risorse

Per ulteriori informazioni, fare clic [qui](https://support.office.com/article/SharePoint-Online-and-OneDrive-for-Business-software-boundaries-and-limits-8f34ff47-b749-408b-abc0-b605e1f6d498).
  
Per informazioni sui costi, fare clic [qui](https://products.office.com/business/compare-office-365-for-business-plans).

## OneDrive for Business

### Caratteristiche

- L'archiviazione è sicura, affidabile e accessibile da più dispositivi e da qualsiasi luogo 
    
- Il contenuto è facilmente individuabile tramite ricerca da parte dell'utente
    
- È possibile accedere al contenuto tramite le API web o un browser oppure non in linea
    
- Ogni utente dispone di 1 TB di spazio di archiviazione
    
### Utilizzi comuni

- Archivio documenti dei singoli utenti
    
- Collaborazione ad hoc nei documenti
    
### Scenari di archiviazione chiave

- File di backup
    
### Risorse

Per ulteriori informazioni, fare clic [qui](https://support.office.com/article/What-is-OneDrive-for-Business-187f90af-056f-47c0-9656-cc0ddca7fdc2).
  
Per informazioni sui costi, fare clic [qui](https://products.office.com/business/compare-office-365-for-business-plans).
  
## Visual Studio Team Services

### Caratteristiche

- Ridondanza incorporata
    
- Basato su Team Foundation Server
    
- Crea backlog, lavora in sprint e tiene traccia di modifiche e bug
    
- L'opzione migliore per l'archiviazione del codice sorgente (è consigliabile archiviare la documentazione in SharePoint Online)
    
### Utilizzi comuni

- Repository di codice sorgente
    
### Scenari di archiviazione chiave

- Gestire il codice sorgente
    
### Risorse

Per ulteriori informazioni, fare clic [qui](https://www.visualstudio.com/products/what-is-visual-studio-online-vs.aspx).
  
Per informazioni sui costi, fare clic [qui](https://www.visualstudio.com/products/visual-studio-online-pricing-vs.aspx).
  
## Condivisione di file Yammer

### Caratteristiche

- Limite di dimensioni del file di 5 GB per ogni file caricato per reti premium
    
- Nessun limite per il numero di file che possono essere caricati
    
- Indicato per condivisione rapida e discussioni
    
- Accessibile tramite un browser
    
### Utilizzi comuni

- Social Enterprise
    
- Condividere documenti o collegamenti relativi a una conversazione o a un argomento
    
### Scenari di archiviazione chiave

- Collabora
    
### Risorse

Per ulteriori informazioni, fare clic [qui](https://products.office.com/yammer/yammer-overview).
  
Per informazioni sui costi, fare clic [qui](https://products.office.com/business/compare-office-365-for-business-plans).
  
## Dynamics 365

### Caratteristiche

- La sottoscrizione include 10 GB di spazio di archiviazione
    
- Per ciascun incremento di 20 Professional USL, vengono aggiunti 5 GB di capacità di archiviazione
    
- Lo spazio di archiviazione è limitato a 5 TB per cliente
    
### Utilizzi comuni

- Archiviare e gestire relazioni con i clienti
    
### Scenari di archiviazione chiave

- Collabora
    
### Risorse

Per ulteriori informazioni, fare clic [qui](http://www.microsoft.com/dynamics/default.aspx?DYNCRM-SEARCH?CR_CC=200470741&amp;WT.mc_ID=DynGB_en_us_SEM_GOOG&amp;DYNCRM-SEARCH&amp;WT.srch=1).
  
## Microsoft Intune

### Caratteristiche

- Tutti i software distribuiti sono compressi e caricati nello spazio di archiviazione cloud di Microsoft Intune
    
- La sottoscrizione include 20 GB di spazio di archiviazione con la possibilità di acquistare ulteriore spazio, vari software e report di inventario hardware, gestione dei criteri, pubblicazione di app per dispositivi mobili, avvisi, monitoraggio e contratti di servizio per un tempo di attività del 99,99%
    
### Utilizzi comuni

- Distribuzione del software
    
### Scenari di archiviazione chiave

- File di backup
    
### Risorse

Per ulteriori informazioni, fare clic [qui](https://www.microsoft.com/it-it/server-cloud/products/microsoft-intune/).
  
Per informazioni sui costi, fare clic [qui](https://www.microsoft.com/it-it/server-cloud/products/microsoft-intune/Purchasing.aspx).
  
## Ripristino del sito di Azure

### Caratteristiche

- Protezione semplice e automatizzata, replica e ripristino per Azure
    
- Monitoraggio continuo dello stato e ripristino orchestrato
    
- Supporta dischi gestiti e macchine virtuali di Ubuntu
    
- Supporta Windows Server 2016 e spazi di archiviazione
    
### Utilizzi comuni

- Le macchine virtuali in esecuzione in un ambiente con Hyper-V possono essere replicate tra due datacenter o in Azure
    
- InMage Scout consente la protezione di VMware e di ambienti fisici
    
### Scenari di archiviazione chiave

- File di backup
    
### Risorse

Per ulteriori informazioni, fare clic [qui](http://azure.microsoft.com/services/site-recovery/?WT.srch=1&amp;WT.mc_ID=kn0iWCHA).
  
Per informazioni sui costi, fare clic [qui](http://azure.microsoft.com/pricing/details/site-recovery/).
  
## Backup di Azure

### Caratteristiche

- Protegge facilmente i dati ricevuti da Windows Server e Windows Server Essentials
    
- Include System Center Data Protection Manager
    
- I dati vengono archiviati in uno spazio di archiviazione geo-replicato, che conserva sei copie dei dati nei due datacenter Azure
    
- Supporta modifiche delta, frequenza definita, compressione, crittografia, conservazione più lunga e limitazione della larghezza di banda
    
### Utilizzi comuni

- Protegge i dati da un'ampia gamma di origini locali (SharePoint, SQL, Exchange) e Azure (VM)
    
### Scenari di archiviazione chiave

- File di backup
    
### Risorse

Per ulteriori informazioni, fare clic [qui](http://azure.microsoft.com/services/backup/).
  
Per informazioni sui costi, fare clic [qui](http://azure.microsoft.com/pricing/details/backup/).
