## <a name="speed-up-page-downloads-by-using-sharepoint-image-renditions"></a><span data-ttu-id="46d0e-133">Velocizzare i download delle pagine tramite rendering delle immagini di SharePoint</span><span class="sxs-lookup"><span data-stu-id="46d0e-133">Speed up page downloads by using SharePoint image renditions</span></span>
