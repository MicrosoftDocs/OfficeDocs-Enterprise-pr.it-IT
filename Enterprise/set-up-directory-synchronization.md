## <a name="office-365-directory-synchronization"></a><span data-ttu-id="da9f2-107">Sincronizzazione delle directory di Office 365</span><span class="sxs-lookup"><span data-stu-id="da9f2-107">Office 365 directory synchronization</span></span>

Questo è lo scenario di sincronizzazione della directory più comune. Autenticazione pass-through o l'identità federate, consente di gestire i propri utenti locali e vengono autenticati dalla directory locale. This is the most common directory synchronization scenario. Pass-through authentication or Federated identity, allows you to manage your users on-premises and they are authenticated by your on-premises directory. Federated identity requires additional configuration and enables your users to only sign in once. For details, read [Understanding Office 365 Identity and Azure Active Directory](about-office-365-identity.md).</span></span>
